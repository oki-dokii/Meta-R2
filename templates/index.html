Key Findings

The Fix That Mattered

Replacing json.loads(full_text) with greedy regex extraction re.search(r'\{.*\}', text, re.DOTALL) caused a 97% improvement within training runs and +85.7% over the base model.

Honest Limitation

Run 4 (v1 final) scored −0.100 — slightly worse than the base model's −0.070 on the same eval. The curriculum plateaued at difficulty 1; the model gained consistency (45/50 non-failing) but not peak reward.

New Capability: Multi-Step

v3's +0.140 episode return is not comparable to the single-step baseline — it measures 3-action sequences with γ=0.9 discounting. The base model has zero multi-step planning capability.
